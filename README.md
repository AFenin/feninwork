# feninwork

Personal website for Alexander Fenin.

## Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui

## Local development

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```

## Deployment

The site is configured for GitHub Pages with a workflow in `.github/workflows/deploy.yml`.
Push to `main` to trigger deployment.
